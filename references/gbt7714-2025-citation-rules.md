# GB/T 7714-2025 参考文献著录规则

> 学习日期：2026-04-07
> 标准编号：GB/T 7714-2025
> 实施日期：2026-07-01（替代 GB/T 7714-2015）

---

## 一、核心修订要点

1. **术语体系调整**
   - 删除：主要责任者、专著、数字对象唯一标识符
   - 新增：会议录、数据集、永久标识符、预印本、责任者
   - 更新：5个术语英文对应词、7个术语定义

2. **文献类型扩充（新增9类）**
   - 会议录、学位论文、报告、标准、网站/网页、档案、地图、数据集、预印本
   - 更名：普通图书→图书，舆图→地图

3. **关键术语变更**
   - "数字对象唯一标识符" → "永久标识符"
   - "公告日期、更新日期" → "创建/发布或修改日期"

---

## 二、术语定义

| 术语 | 定义 |
|------|------|
| 参考文献 | 位于文末或文中，对信息资源进行准确著录的数据，用于识别、定位和检索 |
| 电子资源 | 以数字方式存储，通过计算机或网络使用的信息资源 |
| 会议录 | 包含向大会提交的论文及相关讨论事务的文献 |
| 连续出版物 | 连续发行、计划无限期发行的印刷或非印刷出版物（含期刊、报纸） |
| 数据集 | 相似或相关数据的逻辑集合，通常用于研究 |
| 永久标识符 | 独立于物理位置提供访问，确保数字对象永久存取的唯一标识符（如DOI、URN） |
| 预印本 | 尚未通过同行评审但可广泛使用的文稿 |
| 责任者 | 创造、积累和/或保管信息资源的实体（团体、家族或个人） |

## 三、著录通用规则

### 3.1 著录用符号

| 符号 | 用途 |
|------|------|
| . | 题名、比例尺、版本、出版信息、年卷期、尺寸、访问路径、永久标识符前；每条参考文献结尾 |
| : | 其他题名信息、出版者、页码、学位授予单位、报告编号、专利申请号、档号、收藏者前 |
| , | 同一著作方式的责任者、"等""译"字样、出版年、年和卷号、会议年份、学位授予年前 |
| ; | 同一责任者的合订题名、连续出版物后续的年卷期标识与页码前 |
| // | 析出文献的出处、会议名称前 |
| () | 连续出版物期号/版次、创建/发布或修改日期、非公元纪年出版年 |
| [] | 文献序号、文献类型和载体标识、引用日期、自拟信息 |
| / | 合期/合卷号间、文献载体标识前 |
| - | 起讫页码、起讫序号间 |

### 3.2 责任者著录规则

- 个人责任者：姓在前名在后；欧美责任者名可缩写；汉语拼音姓全大写、名用全称
- 同一著作方式责任者≤3个：全部著录；>3个：著录前3个，后加",等"
- 无责任者：可注明"佚名"（顺序编码制可省略）
- 机关团体：分级著录，上下级间用"."分隔（中文机关团体名称除外）

### 3.3 出版信息规则

- 无出版地：中文著录"[出版地不详]"，外文著录"[S.l.]"
- 无出版者：中文著录"[出版者不详]"，外文著录"[s.n.]"
- 出版年采用公元纪年；其他纪年置于"()"内；估计出版年置于"[]"内
- 电子资源日期格式：YYYY-MM-DD

## 四、各文献类型著录格式

### 4.1 顺序编码制格式

| 文献类型 | 著录格式 |
|----------|----------|
| 图书 | 主要责任者.题名：其他题名信息[文献类型标识/载体标识].其他责任者.版本.出版地：出版者，出版年：引文页码.访问路径.永久标识符. |
| 图书析出文献 | 析出文献主要责任者.析出文献题名：其他题名信息[文献类型标识/载体标识].析出文献其他责任者//图书主要责任者.图书题名：其他题名信息.版本.出版地：出版者，出版年：析出页码.访问路径.永久标识符. |
| 连续出版物析出文献 | 析出文献主要责任者.析出文献题名：其他题名信息[文献类型标识/载体标识].析出文献其他责任者.连续出版物题名：其他题名信息，年，卷(期/版次):页码.访问路径.永久标识符. |
| 会议录 | 主要责任者.题名：其他题名信息[C/载体标识]//会议名称，会议年份：引文页码.访问路径.永久标识符. |
| 学位论文 | 主要责任者.题名：其他题名信息[D/载体标识].学位授予单位所在地：学位授予单位，学位授予年：引文页码.访问路径.永久标识符. |
| 报告 | 主要责任者.题名：其他题名信息：报告编号[R/载体标识].发布日期：引文页码.访问路径.永久标识符. |
| 标准 | 标准编号 标准名称[S/载体标识].访问路径.永久标识符. |
| 专利 | 专利申请者/所有者.题名：其他题名信息：专利申请号[P/载体标识].公告(公开)日期：引文页码.访问路径.永久标识符. |
| 网页 | 主要责任者.题名：其他题名信息[EB/OL].(创建或修改日期)[引用日期].访问路径.永久标识符. |
| 档案 | 主要责任者.题名：其他题名信息：档号[A/载体标识].收藏者所在地：收藏者，形成日期：引文页码.访问路径.永久标识符. |
| 地图 | 主要责任者.题名：其他题名信息.比例尺[CM/载体标识].版本.出版地：出版者，出版年.尺寸.访问路径.永久标识符. |
| 数据集 | 主要责任者.题名：其他题名信息[DS/载体标识].版本.发布平台(发布或修改日期)[引用日期].访问路径.永久标识符. |
| 预印本 | 主要责任者.题名：其他题名信息[PP/OL].版本.出版平台(创建或修改日期)[引用日期].访问路径.永久标识符. |

### 4.2 著者-出版年制调整

将出版年置于责任者后，以"."分隔。参考文献表按责任者字顺和出版年排序。

## 五、参考文献标引体系

### 5.1 顺序编码制

**正文引用：**
- 按文献出现先后顺序连续编码，序号置于"[]"中
- 同一处引用多篇：序号在"[]"内用","分隔；连续序号用"-"连接
- 多次引用同一文献：标注首次引用序号，序号外著录引文页码

**参考文献表：** 按正文标注序号依次排列

### 5.2 著者-出版年制

**正文引用：**
- 标注内容：责任者姓氏与出版年，置于"()"内
- 正文中已提及责任者：仅著录出版年
- 多责任者文献：标注第一责任者后加"等"（中文）或"et al."（外文）
- 同一责任者同年多篇：出版年后加小写字母a、b、c区分
- 多次引用同一文献：在"()"外角标著录引文页码

**参考文献表：** 先按文种集中，再按责任者字顺和出版年排列

## 七、关键变化速查

| 项目 | 2015版 | 2025版 |
|------|--------|--------|
| 预印本 | 无 | 新增，代码PP |
| 网页类型代码 | EB | EB（但文献类型调整为"网站、网页"） |
| 数字标识符 | 数字对象唯一标识符 | 永久标识符 |
| 日期表述 | 公告日期、更新日期 | 创建/发布或修改日期 |
