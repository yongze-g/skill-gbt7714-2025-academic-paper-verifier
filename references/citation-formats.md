## 概览

本文档详细说明 GB/T 7714-2025《信息与文献 参考文献著录规则》中各文献类型的著录格式。

**必备性定义**：
- **必备**：必须著录，缺失则文献信息不完整
- **有则必备**：如有该信息则必须著录，无则可省略
- **可选**：根据情况决定是否著录

**著录用符号必须逐条核实**：. : , ; // [] ()

## 著录用符号规则

著录用符号及其前置位置（必须逐条核实）：

| 符号 | 名称 | 前置位置 |
|------|------|----------|
| `.` | 点 | 题名、比例尺、版本、出版信息、年卷期、访问路径、永久标识符前 |
| `:` | 冒号 | 其他题名信息、出版者、页码、学位授予单位、报告编号、专利申请号、档号、收藏者前 |
| `,` | 逗号 | 同一著作方式的责任者、"等""译"字样、出版年、年和卷号、会议年份、学位授予年前 |
| `;` | 分号 | 同一责任者的合订题名、连续出版物后续的年卷期标识与页码前 |
| `//` | 双斜杠 | 析出文献的出处、会议名称前 |
| `[]` | 方括号 | 文献序号、文献类型和文献载体标识、引用日期、自拟信息 |
| `()` | 圆括号 | 连续出版物及其析出文献中的期号/版次、创建/发布或修改日期、非公元纪年 |

**禁止**：不按类型要求堆砌所有信息，不按必备性著录。

## 会议录 [C]

**著录项目（必备性）：**
- 主要责任者 [有则必备]
- 题名 [必备]
- 文献类型标识 [必备]
- 文献载体标识 [电子资源必备]
- 会议名称 [有则必备]
- 会议年份 [有则必备]
- 引文页码 [有则必备]
- 获取和访问路径 [电子资源必备]
- 永久标识符 [电子资源可选]

**格式模板：**
```
主要责任者. 题名[文献类型标识/文献载体标识]//会议名称, 会议年份: 引文页码. 获取和访问路径. 永久标识符.
```

**示例（纸质）：**
```
[1] 李妍, 王莹. 医疗机构保洁人员"一前五后"手卫生干预效果研究[C]//中华预防医学会医院感染控制分会第31次全国医院感染学术年会, 2022: 2.
```

**示例（电子资源）：**
```
[1] Yu Yang, Pan Erting, Wang Xinya, et al. Unmixing before fusion: a generalized paradigm for multi-source based hyperspectral image synthesis[C/OL]//CVPR, 2024: 4. https://openaccess.thecvf.com/content/CVPR2024/html/Yu_Unmixing_Before_Fusion_A_Generalized_Paradigm_for_Multi-Source-based_Hyperspectral_Image_CVPR_2024_paper.html.
[2] Wang Shansan. Application of improved SOM neural network in intelligent auditing of hospital financial vouchers[C/OL]//2022 6th Asian Conference on Artificial Intelligence Technology, 2022: 2. https://ieeexplore.ieee.org/document/10137867. DOI:10.1109/ACAIT56212.2022.10137867.
```

**注意：** 会议录**不需要**出版地、出版者信息。会议名已含年份时，后面不再重复年份。

```

[2] 肖玲, 张雪, 王永. 数据要素的统计测算方法探究[PP/OL]. PSSXiv(2024-07-02)[2024-09-30]. https://zsyyb.cn/abs/202408.01096.
```
