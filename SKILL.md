# 学术文献验证

## 概述

本技能用于验证学术文献的准确性，包括确认论文是否存在、核实关键信息（作者、标题、发表源、年份等），并依据 GB/T 7714-2025《信息与文献 参考文献著录规则》生成规范的参考文献著录格式。

**核心原则：专事专办，按文献类型匹配对应要求，不堆砌信息。**

## 工作流程

### 步骤1：识别文献类型

判断文献所属类型，常见类型包括：
- 期刊论文 [J]
- 会议录（会议论文） [C]
- 图书（专著） [M]
- 学位论文 [D]
- 报告 [R]
- 标准 [S]
- 专利 [P]
- 预印本 [PP]
- 电子资源（网页、数据库等） [EB/OL], [DB/OL]

### 步骤2：查阅著录项目表

根据文献类型，查阅 GB/T 7714-2025 对应章节的著录项目表，确定：
- **必备项目**：必须著录的信息
- **可选项目**：有则著录、无则省略的信息

**禁止**：不按类型要求堆砌所有信息。

### 步骤3：信息收集

使用网络搜索获取该类型要求的必备信息：

```
web_search 搜索关键词：论文标题 + 作者 + 发表年份
```

**搜索目标（按类型筛选）：**

| 文献类型 | 必备信息 |
|----------|----------|
| 期刊论文 [J] | 作者、题名、刊名、年、卷(期)、页码 |
| 会议录 [C] | 作者、题名、会议名称、会议年份、引文页码；电子资源需获取和访问路径 |
| 图书 [M] | 主要责任者、题名、出版地、出版者、出版年、引文页码 |
| 学位论文 [D] | 学位论文作者、题名、学位授予单位、年份 |
| 预印本 [PP] | 主要责任者、题名、预印本平台、发布时间、永久标识符 |

### 步骤4：信息核实与表格输出

整理核实信息，**必须输出核实信息表**：

| 字段 | 类型要求 | 论文信息 | 是否必备 |
|------|----------|----------|----------|
| 主要责任者 | 有则必备 | 作者名 | 是/否 |
| 题名 | 必备 | 论文标题 | 是 |
| 文献类型标识 | 必备 | [J]/[C]/[M]等 | 是 |
| ... | ... | ... | ... |

**作者姓名著录规则（GB/T 7714-2025 7.1.1）：**

| 人名类型 | 姓 | 名 | 示例 |
|----------|-----|-----|------|
| 中国人名（汉语拼音） | 全部著录，首字母大写 | **宜用全称**，每个字首字母大写 | `Yu Yang`, `Wang Xinya` |
| 欧美人名（西文） | 全部著录，首字母大写 | 可缩写为首字母（如无法识别时著录全名） | `Vaswani A`, `Einstein A`, `Godard J-L` |

**关键区分：**
- 中国人名（汉语拼音）：**名用全称**（Yang，非Y）
- 欧美人名（西文）：**名可缩写**（A.）
- 同姓不同名的欧美人：名的首字母也应著录以区分

**作者数量规则：**
- 3人及以下：全部列出
- 4人及以上：列出前3人后加 `, et al.` 或 `, 等`

## 类型标识与获取路径统一原则（v1.1.0 新增）

### 核心原则

**文献类型标识必须与获取路径保持一致。**

| 文献类型 | 类型标识 | 获取路径要求 | 常见来源 |
|----------|----------|--------------|----------|
| 会议录（电子资源） | `[C/OL]` | **必须附带会议官方链接** | OpenReview, IEEE Xplore, PMLR, CVF |
| 预印本 | `[PP/OL]` | 附带预印本平台链接 | arXiv, bioRxiv, SSRN |
| 期刊（电子资源） | `[J/OL]` | 附带期刊官网或DOI链接 | PubMed, Wiley, Springer |

### 降级处理规则

**会议链接无法获取时的处理流程：**

```
尝试获取会议官方链接
    ↓ 失败
自动降级为 [PP/OL] 类型
    ↓
使用预印本链接（arXiv等）
    ↓
在查证链接表中标注 ⚠️ 无会议链接
```

**示例：**

| 原格式（错误） | 降级后格式（正确） |
|--------------|------------------|
| `[C/OL]//ICLR, 2015. https://arxiv.org/abs/1412.6980` | `[PP/OL]. arXiv, 2014-12-22. https://arxiv.org/abs/1412.6980` |

### 常见会议官方链接来源

| 会议 | 官方链接来源 | 说明 |
|------|-------------|------|
| ICLR | OpenReview (openreview.net) | 会议论文集 |
| NeurIPS | NeurIPS官网 或 OpenReview | 会议论文集 |
| ICML | PMLR (proceedings.mlr.press) | 会议论文集 |
| CVPR/ICCV | CVF Open Access (openaccess.thecvf.com) | 会议论文集 |
| ACL/EMNLP | ACL Anthology (aclanthology.org) | 会议论文集 |
| IEEE会议 | IEEE Xplore (ieeexplore.ieee.org) | 会议论文集 |

---

### 步骤5：引用格式生成

依据 GB/T 7714-2025 生成两种引用体系的格式。

**顺序编码制示例（会议录）：**
```
[1] Vaswani A, Shazeer N, Parmar N, et al. Attention is all you need[C/OL]//NIPS 2017, 2017: 5998-6008. https://doi.org/10.48550/arXiv.1706.03762.
```

**著者-出版年制示例（会议录）：**
```
Vaswani A, Shazeer N, Parmar N, et al. 2017. Attention is all you need[C/OL]//NIPS 2017, 2017: 5998-6008. https://doi.org/10.48550/arXiv.1706.03762.
```

---

### 步骤6：输出规范（v1.1.0 更新）

**必须输出两部分：**

#### 6.1 纯文本引用格式

```
[1] 作者. 题名[类型标识]//来源, 年份: 页码. 获取路径.
[2] ...
```

#### 6.2 人工查证链接表

| 序号 | 文献题名 | 查证链接 | 链接类型 | 备注 |
|:----:|----------|----------|----------|------|
| 1 | 题名 | [链接](URL) | 会议官方 | 正常 |
| 2 | 题名 | [链接](URL) | 预印本 | ⚠️无会议链接 |
| 3 | 题名 | [链接](URL) | 期刊官网 | 正常 |
|  |  | [链接](URL) | DOI | - |

**表格说明：**
- 一个文献可有多条链接（会议官网 + 预印本 + DOI）
- 链接类型包括：会议官方、预印本、期刊官网、DOI、数据库
- 备注栏标注异常情况（如无会议链接、需VPN等）

## 各类型著录格式速查

### 会议录 [C]

**必备项目：** 主要责任者、题名、会议名称、会议年份、引文页码

**电子资源增加：** 获取和访问路径

**格式模板：**
```
主要责任者. 题名[文献类型标识/文献载体标识]//会议名称，会议年份：引文页码. 获取和访问路径. 永久标识符.
```

**示例（纸质）：**
```
[1] 李妍, 王莹. 医疗机构保洁人员"一前五后"手卫生干预效果研究[C]//中华预防医学会医院感染控制分会第31次全国医院感染学术年会, 2022: 2.
```

**示例（电子资源）：**
```
[1] Yu Yang, Pan Erting, Wang Xinya, et al. Unmixing before fusion: a generalized paradigm for multi-source based hyperspectral image synthesis[C/OL]//CVPR, 2024: 4. https://openaccess.thecvf.com/content/CVPR2024/html/Yu_Unmixing_Before_Fusion_A_Generalized_Paradigm_for_Multi-Source-based_Hyperspectral_Image_CVPR_2024_paper.html.
[2] Wang Shansan. Application of improved SOM neural network in intelligent auditing of hospital financial vouchers[C/OL]//2022 6th Asian Conference on Artificial Intelligence Technology, 2022: 2. https://ieeexplore.ieee.org/document/10137867. DOI:10.1109/ACAIT56212.2022.10137867.
```

**注意：** 会议录**不需要**出版地、出版者信息。会议名已含年份时，后面不再重复年份。

### 预印本 [PP]

**必备项目：** 主要责任者、题名、预印本平台、发布时间、永久标识符

**格式模板：**
```
主要责任者. 题名[文献类型标识/文献载体标识]. 预印本平台, 发布时间. 永久标识符.
```

**示例：**
```
[1] Kingma D P, Ba J L. Adam: A method for stochastic optimization[PP/OL]. arXiv, 2014-12-22. https://arxiv.org/abs/1412.6980.
```

### 期刊论文 [J]

**必备项目：** 主要责任者、题名、刊名、年、卷(期)、页码

**格式模板：**
```
主要责任者. 题名[文献类型标识]. 刊名, 年, 卷(期): 页码.
```

**示例（纸质）：**
```
[1] 于潇, 刘义, 柴跃廷, 等. 互联网药品可信交易环境中主体资质审核备案模式[J]. 清华大学学报(自然科学版), 2012, 52(11): 1518-1523.
[2] Saito M, Miyazaki K. Jadeite-bearing metagabbro in serpentinite melange of the "Kurosegawa Belt" in Izumi Town, Yatsushiro City, Kumamoto Prefecture, central Kyushu[J]. Bulletin of the Geological Survey of Japan, 2006, 57(5/6): 169-176.
```

**示例（电子资源）：**
```
[1] Veen P Hvd, Muller M, Vincken K L, et al. Longitudinal changes in brain volumes and cerebrovascular lesions on MRI in patients with manifest arterial disease: the SMART-MR study[J/OL]. J Neurol Sci, 2014, 337(1/2): 112-118. https://pubmed.ncbi.nlm.nih.gov/24314719/. DOI:10.1016/j.jns.2013.11.029
[2] Maurya A, Szymanski M, Karlowski W M. ARA: a flexible pipeline for automated exploration of NCBI SRA datasets[J/OL]. GigaScience, 2023, 12: giad067. https://doi.org/10.1093/gigascience/giad067.
```

### 图书（专著）[M]

**必备项目：** 主要责任者、题名、出版地、出版者、出版年、引文页码

**格式模板：**
```
主要责任者. 题名[文献类型标识]. 出版地: 出版者, 出版年: 引文页码.

**示例（纸质）：**

**示例（电子资源）：**
```
[1] 赵学功. 当代美国外交[M/OL]. 北京: 社会科学文献出版社, 2001. http://www.cadal.zju.edu.cn/book/trySinglePage/33023884/1.

## 批量处理规范

当用户一次提供多条参考文献时：

### 步骤1：总览表格

先输出总览表格，列出所有条目：

| 序号 | 题名 | 文献类型 | 状态 |
|------|------|----------|------|
| 1 | Attention is all you need | 会议录[C] | 待核实 |
| 2 | Deep learning | 期刊[J] | 待核实 |
| ... | ... | ... | ... |

### 步骤2：逐条拆解处理

对每条文献分别执行：
1. 识别文献类型
2. 查阅著录项目表
3. 收集必备信息
4. 输出核实信息表
5. 生成引用格式

### 步骤3：汇总输出

最后汇总所有引用格式和人工查证链接表。

## 常见错误警示

| 错误类型 | 错误示例 | 正确做法 |
|----------|----------|----------|
| 作者大小写（西文） | VASWANI A（全大写） | Vaswani A（姓首字母大写，名缩写） |
| 中国人名缩写 | Yu Y（名缩写） | Yu Yang（**名用全称**） |
| 年份重复 | `//CVPR 2024, 2024`（会议名已含年份） | `//CVPR, 2024`（会议名不含年份时才加逗号年份） |
| 会议录信息冗余 | 添加出版地、出版者 | 会议录只需会议名、年份、页码 |
| 格式套用错误 | 用图书析出格式著录会议录 | 按文献类型选对应格式 |
| 信息堆砌 | 把所有能查到的信息都写上 | 只写该类型要求的必备项 |
| **类型与路径不一致（v1.1.0）** | `[C/OL]` 附带 arXiv 链接 | 应降级为 `[PP/OL]` 或改用会议官网链接 |
