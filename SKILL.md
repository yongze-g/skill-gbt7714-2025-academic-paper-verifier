# 学术文献验证

## 概述

本技能用于验证学术文献的准确性，包括确认论文是否存在、核实关键信息（作者、标题、发表源、年份等），并依据 **GB/T 7714-2025《信息与文献 参考文献著录规则》** 生成规范的参考文献著录格式。

**核心原则：专事专办，按文献类型匹配对应要求，不堆砌信息。**

---

## 工作流程

### 步骤1：分析指令

首先分析用户发来的指令：
- **单条参考文献**：直接按后续步骤处理
- **多条参考文献**：制定计划，逐条依次处理

### 步骤2：处理单条参考文献

#### 2.1 提取关键字

从用户提供的参考文献信息中提取关键字：
- 作者（主要责任者）
- 题名（标题）
- 发表源（期刊/会议/出版社等）
- 发表年份
- 其他已知信息（卷期、页码等）

#### 2.2 信息收集

使用网络搜索获取该文献的完整信息：

```
web_search 搜索关键词：论文标题 + 作者 + 发表年份
```

**搜索策略**：
- 优先使用精确匹配搜索
- 如无法获取，适当使用模糊搜索
- 尝试多个搜索引擎和学术数据库

#### 2.3 信息匹配验证

判断搜索结果是否与用户提供的文献一致：
- 比对作者姓名（注意拼写变体）
- 比对题名（注意中英文差异）
- 比对发表年份和来源
- 注意排除拼写相近但内容不同的文献

#### 2.4 识别文献类型与载体标识

根据搜索结果识别文献类型和载体类型，详细的标识代码见 `references/gbt7714-2025-core-rules.md`，速查见本文档末尾的**文献类型标识速查**。

#### 2.5 获取引用格式模板

根据文献类型和载体标识，从 `references/citation-formats.md` 中获取对应的格式模板：

**示例格式模板：**

| 文献类型 | 格式模板 |
|----------|----------|
| 期刊论文[J/OL] | 主要责任者. 题名: 其他题名信息[J/OL]. 刊名, 年, 卷(期): 页码. 获取路径. 永久标识符. |
| 会议录[C/OL] | 主要责任者. 题名: 其他题名信息[C/OL]//会议名称, 会议年份: 引文页码. 获取路径. 永久标识符. |
| 预印本[PP/OL] | 主要责任者. 题名: 其他题名信息[PP/OL]. 版本. 出版平台(创建日期)[引用日期]. 获取路径. 永久标识符. |

#### 2.6 准备必备项表格

根据文献类型要求，整理必备项信息：

| 字段 | 类型要求 | 文献信息 | 是否必备 |
|------|----------|----------|----------|
| 主要责任者 | 有则必备 | 作者名 | 是/否 |
| 题名 | 必备 | 论文标题 | 是 |
| 文献类型标识 | 必备 | [J]/[C]/[M]等 | 是 |
| 文献载体标识 | 电子资源必备 | /OL等 | 是/否 |
| 刊名/会议名/出版者 | 必备 | 发表来源 | 是 |
| 年 | 必备 | 发表年份 | 是 |
| 卷/期/页码 | 有则必备 | 卷期页码 | 是/否 |
| 获取路径 | 电子资源必备 | URL/DOI | 是/否 |
| 永久标识符 | 电子资源可选 | DOI等 | 否 |

#### 2.7 常见问题排查

根据 `references/troubleshooting.md` 中的常见错误警示，排查必备项：

| 检查项 | 检查内容 | 处理方式 |
|--------|----------|----------|
| 作者姓名 | 中国人名用全称，欧美人名可缩写 | 修正格式 |
| 年份重复 | 会议名已含年份时不再重复 | 删除重复年份 |
| 类型路径一致性 | 文献类型标识与获取路径匹配 | 不匹配时降级处理 |
| 信息冗余 | 不添加类型要求外的信息 | 删除冗余信息 |

**排查结果处理：**
- 发现问题 → 回滚并重新收集信息
- 未发现问题 → 进入下一步

#### 2.8 生成引用格式

将纠错后的必备项填入格式模板，生成规范的引用格式：

**顺序编码制示例：**
```
[1] Vaswani A, Shazeer N, Parmar N, et al. Attention is all you need[C/OL]//NIPS 2017, 2017: 5998-6008. https://doi.org/10.48550/arXiv.1706.03762.
```

### 步骤3：汇总输出

当处理多条参考文献时，汇总所有引用：

#### 3.1 纯文本引用格式

```
[1] 作者1. 题名1[类型标识]//来源1, 年份1: 页码1. 获取路径1.
[2] 作者2. 题名2[类型标识]//来源2, 年份2: 页码2. 获取路径2.
...
```

#### 3.2 人工查证链接表

| 序号 | 文献题名 | 查证链接 | 链接类型 | 备注 |
|:----:|----------|----------|----------|------|
| 1 | 题名1 | [链接1](URL) | 会议官方/期刊官网/预印本 | 正常/⚠️无会议链接 |
| 2 | 题名2 | [链接2](URL) | DOI | - |
| ... | ... | ... | ... | ... |
