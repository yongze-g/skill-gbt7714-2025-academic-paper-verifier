# Changelog

所有版本的变更记录均在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.2.0] - 2026-04-14

### 新增
- **文献匹配性校验机制**：新增步骤3.5，确保至少3个独立信息项完全匹配才认定文献相同
- **模糊信息警示**：标题不完整、拼写错误、作者/年份/发表源缺失时发出警示
- **格式逐条核验流程**：在生成引用格式前对照常见错误警示表逐项检查
- **新增6种文献类型支持**：档案[A]、地图[CM]、数据集[DS]、报纸[N]、图书中的析出文献[M]、连续出版物中的析出文献[J]/[N]
- **预印本格式规范**：明确预印本[PP/OL]格式要求：`平台(创建或修改日期)[引用日期]`
- **arXiv文献识别**：重点强调arXiv等平台文献必须识别为预印本[PP/OL]
- **详细著录格式文档**：新增 references/citation-formats.md，包含15种文献类型的完整著录格式和示例
- **GB/T 7714-2025完整规则**：大幅完善 references/gbt7714-2025-citation-rules.md，包含14个著录项目表

### 优化
- **输出结构精简**：移除修改汇总表输出，仅保留最终引用格式和查证链接表
- **文档结构重构**：SKILL.md从746行精简至377行，符合≤500行规范
- **必备性分类**：明确必备/有则必备/可选三级分类，每个著录项目标注必备性
- **著录用符号规则**：详细说明7种符号的精确前置位置和使用规则
- **汉语拼音作者著录**：明确姓首字母大写，名用全称（如Li Jiangning）
- **同姓不同名处理**：著录名的首字母或全名以区分责任者

### 修复
- 修复预印本格式错误：从`arXiv, 2014-12-22`修正为`arXiv(2014-12-22)[2024-09-30]`
- 修正汉语拼音作者格式：从`YU Yang`（姓全部大写）修正为`Li Jiangning`（姓首字母大写）
- 修复中国人名错误提示：将"中国人名缩写"改为"汉语拼音作者格式错误"
- 修正责任者著录规则说明：明确区分"西文著录"和"汉语拼音著录"，不限定国籍

### 变更
- **术语更新**："数字对象唯一标识符" → "永久标识符"
- **类型判断优先级**：预印本平台识别提升到第2位
- **著录符号说明**：从步骤2提取到附录，增加详细速查表
- **参考文档结构**：新增两层参考文档（citation-formats.md + gbt7714-2025-citation-rules.md）

## [1.1.0] - 2026-04-08

### 新增
- **类型标识与获取路径统一原则**：文献类型标识必须与获取路径保持一致
  - `[C/OL]` 类型必须附带会议官方链接
  - `[PP/OL]` 类型附带预印本链接（arXiv等）
  - 会议链接无法获取时自动降级为 `[PP/OL]`
- **人工查证链接表**：输出格式新增表格，列出可点击的查证链接
- **降级处理规则**：会议信息无法核实时，自动降级为预印本类型
- **输出规范更新**：必须输出纯文本引用 + 人工查证链接表（含报错信息）

### 修复
- 修复 `[C/OL]` 类型错误附带 arXiv 链接的问题
- 修复会议录与预印本类型混淆的问题

## [1.0.0] - 2026-04-08

### 新增
- 初始版本发布
- 支持13种文献类型的著录格式生成：
  - 期刊论文 [J]
  - 会议录 [C]
  - 图书（专著）[M]
  - 学位论文 [D]
  - 报告 [R]
  - 标准 [S]
  - 专利 [P]
  - 报纸 [N]
  - 预印本 [PP]
  - 电子公告/网页 [EB/OL]
  - 在线数据库 [DB/OL]
  - 图书中的析出文献
  - 连续出版物中的析出文献
- 完整实现 GB/T 7714-2025 标准规则
- 支持两种引用体系：顺序编码制、著者-出版年制
- 添加31个国标示例（中英文对照，纸质与电子资源）
- 详细的作者姓名著录规则（中国人名vs欧美人名）
- 常见错误警示表

### 技术特性
- 基于网络搜索自动获取文献信息
- 多源交叉验证确保信息准确性
- 批量处理支持
- 信息核实表格输出
