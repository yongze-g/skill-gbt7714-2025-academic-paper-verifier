# Changelog

所有版本的变更记录均在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.0.0] - 2026-04-08

### 新增
- 初始版本发布
- 支持13种文献类型的著录格式生成：
  - 期刊论文 [J]
  - 会议录 [C]
  - 图书（专著）[M]
  - 学位论文 [D]
  - 报告 [R]
  - 标准 [S]
  - 专利 [P]
  - 报纸 [N]
  - 预印本 [PP]
  - 电子公告/网页 [EB/OL]
  - 在线数据库 [DB/OL]
  - 图书中的析出文献
  - 连续出版物中的析出文献
- 完整实现 GB/T 7714-2025 标准规则
- 支持两种引用体系：顺序编码制、著者-出版年制
- 添加31个国标示例（中英文对照，纸质与电子资源）
- 详细的作者姓名著录规则（中国人名vs欧美人名）
- 常见错误警示表

### 技术特性
- 基于网络搜索自动获取文献信息
- 多源交叉验证确保信息准确性
- 批量处理支持
- 信息核实表格输出
