# Changelog

所有版本的变更记录均在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.1.0] - 2026-04-08

### 新增
- **类型标识与获取路径统一原则**：文献类型标识必须与获取路径保持一致
  - `[C/OL]` 类型必须附带会议官方链接
  - `[PP/OL]` 类型附带预印本链接（arXiv等）
  - 会议链接无法获取时自动降级为 `[PP/OL]`
- **人工查证链接表**：输出格式新增表格，列出可点击的查证链接
- **降级处理规则**：会议信息无法核实时，自动降级为预印本类型
- **输出规范更新**：必须输出纯文本引用 + 人工查证链接表（含报错信息）

### 修复
- 修复 `[C/OL]` 类型错误附带 arXiv 链接的问题
- 修复会议录与预印本类型混淆的问题

## [1.0.0] - 2026-04-08

### 新增
- 初始版本发布
- 支持13种文献类型的著录格式生成：
  - 期刊论文 [J]
  - 会议录 [C]
  - 图书（专著）[M]
  - 学位论文 [D]
  - 报告 [R]
  - 标准 [S]
  - 专利 [P]
  - 报纸 [N]
  - 预印本 [PP]
  - 电子公告/网页 [EB/OL]
  - 在线数据库 [DB/OL]
  - 图书中的析出文献
  - 连续出版物中的析出文献
- 完整实现 GB/T 7714-2025 标准规则
- 支持两种引用体系：顺序编码制、著者-出版年制
- 添加31个国标示例（中英文对照，纸质与电子资源）
- 详细的作者姓名著录规则（中国人名vs欧美人名）
- 常见错误警示表

### 技术特性
- 基于网络搜索自动获取文献信息
- 多源交叉验证确保信息准确性
- 批量处理支持
- 信息核实表格输出
